Questionnaire Maker

Create Survey - JSON

If you have a survey created with JSON, which matches the accepted format, then navigate to the Upload JSON page using the navigation. Either upload the JSON file directly, or copy and paste it into the textbox. You will get the chance to view what your survey looks like before you create it.

Create Survey - GUI

If you wish to create a survey from scratch, visit the Create Survey page using the navigation. Use the buttons to add different types of questions, providing a unique name for each.

Answer Survey

You can answer a survey by using the unique survey link provided.

View Survey Results

You can view the results of a survey by visiting the unique results page for each survey. You can view the data as a basic table, or as 4 different styles of charts.

List of Surveys

To view a list of all surveys navigate to the Surveys page using the navigation. On this page you are able to see all surveys created, along with the links to their answer and results page.

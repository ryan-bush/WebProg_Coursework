Accepted JSON Format

Example Usage


{
  "name": "Example Questionnaire",
  "questions": [
    {
      "id": "name",
      "text": "What is your name?",
      "type": "text"
    },
    {
      "id": "quest",
      "text": "What is your quest?",
      "type": "text"
    },
    {
      "id": "col",
      "text": "What is your favourite colour?",
      "type": "text"
    },
    {
      "id": "velo",
      "text": "What is the air-speed velocity of an unladen swallow?",
      "type": "number"
    },
    {
      "id": "lord",
      "text": "Which is the best lord?",
      "type": "single-select",
      "options": [
        "Lord of the Rings",
        "Lord of the Flies",
        "Lord of the Dance",
        "Lorde"
      ]
    },
    {
      "id": "langs",
      "text": "Which computer languages have you used?",
      "type": "multi-select",
      "options": [
        "JavaScript",
        "Java",
        "C",
        "Python",
        "Ook",
        "LISP"
      ]
    }
  ]
}

Name

Name of your survey

Questions

All of your survey questions with the following format.

id

This is the unique id of the question. There must not be another question with the same id.

text

The question you wish to ask.

type

The type of question from the following:

  • text - A question that allows any user input.
  • number - A question that allows any numeric.
  • single-select - A question that allows a single option to be selected.
  • multi-select - A question that allows multiple options to be selected.

options

Only valid for single-select or multi-select questions. All the options the user can select from.
